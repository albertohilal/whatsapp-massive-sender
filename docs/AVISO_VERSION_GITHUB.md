# AVISO DE VERSIÓN

Esta versión del proyecto whatsapp-massive-sender contiene la nueva lógica de sincronización y normalización de las tablas:
- `ll_lugares`
- `ll_lugares_haby`
- `llxbx_societe`
- `ll_lugares_clientes`

Incluye los scripts:
- `recrear_ll_lugares_clientes.js` (nueva lógica de repoblado)
- `diagnostico_sincronizacion.js` (diagnóstico final)
- `RESUMEN_SINCRONIZACION_TABLAS.md` (documentación y resultados)

**IMPORTANTE:**
Esta versión NO está desplegada ni sincronizada en el servidor Contabo. Es una versión avanzada y actualizada, exclusiva de este repositorio en GitHub.

Para auditoría y trazabilidad, consultar el archivo `RESUMEN_SINCRONIZACION_TABLAS.md`.
